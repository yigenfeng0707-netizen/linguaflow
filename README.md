# LinguaFlow - 多语种在线教育平台

沉浸式语言学习体验，支持英语、日语、韩语等主流语言学习。

## 功能特性

- 🎯 **分级课程体系** - 从A1到C2，科学分级
- 📚 **互动式学习** - 单词记忆、语法练习、口语跟读、听力训练
- 📊 **学习进度追踪** - 可视化学习数据
- 👤 **用户认证** - 注册、登录、第三方登录
- 🎓 **个性化推荐** - 基于学习数据的智能推荐
- 💬 **社区交流** - 学习心得分享
- 🏆 **成就激励** - 徽章、排行榜系统

## 技术栈

- Next.js 14 + TypeScript
- Tailwind CSS
- Supabase (PostgreSQL + Auth)
- Docker & Docker Compose

## 快速开始

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env

# 启动开发服务器
npm run dev
```
